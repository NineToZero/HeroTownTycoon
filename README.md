## ✏️ **게임 소개**
 
</div>

- 게임 이름 : 용사마을 타이쿹

- 게임 컨셉

   **`농사 및 제작 시뮬레이션`** + **`성장시킨 동료를 통한 전투 시뮬레이션`**

- 아트 컨셉

   **`Paper2D`** : 캐릭터, 농작물 등의 상호작용 오브젝트

   **`Voxel`** : 건물과 지형지물 오브젝트

- 조작 컨셉

   **`TPS`** : 시점을 마우스로 돌릴 수 있도록 하여 **Paper2D**의 아트 컨셉 극대화

   **`건물`** : 게임 내 주요 기능들은 대부분 건물을 통하여 체험 가능, 깔끔한 정렬을 위해 **셀 그리드**를 사용함
